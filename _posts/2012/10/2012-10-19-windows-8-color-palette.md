So Windows 8 has a new custom Start screen that allows live tiles to be displayed and rearranged on a per user basis. They also allow complete customization of the new Start screen. This includes a few backgrounds that are updated based on the color choice the user selects. Each color choice comes with two colors â€“ a default color for the live tile and a background color for the start menu.

The color choices resemble the following image:

[![windows8-colors](http://jasongaylord.com/Media/Default/Windows-Live-Writer/Windows-8-Color-Palette_DF2F/windows8-colors_thumb.jpg "windows8-colors")](http://jasongaylord.com/Media/Default/Windows-Live-Writer/Windows-8-Color-Palette_DF2F/windows8-colors_2.jpg)
